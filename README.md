# OrgChart Angular

Implementação do Balkan OrgChartJS utilizando Angular

## Instalação de dependências
```
npm install
```

## Executar a aplicação

Executar app + api no mesmo terminal
```
npm run server
```

Executar app + api em terminais separados
```
npm start
npm run api
```
